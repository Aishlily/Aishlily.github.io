CSS知识梳理--定位浮动

发表于 2018-01-04 | 分类于 CSS技术

定位position

position取值：static、relative、absolute、fixed，static是默认值
坐标取值：left、right、top、bottom

相对定位：relative

  1. relative会导致自身位置的相对变化，但是不会脱离文档流，不影响其他元素的位置、大小
  2. relative元素根据自身位置定位

绝对定位：absolute

  1. absolute元素会脱离文档流，同float会导致父元素坍塌
  2. absolute元素具有“包裹性”，同float
  3. absolute元素层级高于普通流元素，会遮挡非absolute元素
  4. absolute元素根据父元素中最近的position:relative/absolute/fixed元素确定位置

固定定位：fixed

同absolute，区别在于，fixed根据window/iframe确定位置，而absolute元素根据父元素中最近的定位元素确定位置

浮动float

float取值： left、right、none

什么时候用浮动

  • 文字环绕效果

破坏性

float元素会脱离文档流。float会破坏父标签的结构，使得服标签出现坍塌现象

包裹性

没有设置宽度的div会撑满整个父容器。添加float:left之后，div的宽度会自动调整为内容的宽度，而不是撑满这个父容器。（此时，display:block没有变化）

清空格

因为多个img标签有换行，img中间会有空格。增加float:left样式之后，空格会消失。因为float元素是脱离文档流的，所以换行、空格都没有了

Float = block

浮动元素会自动被应用一个 display: block 规则，和块元素行为接近（除了占据整行宽度）：

  • 你可以为浮动元素设置宽和高
  • 如果没有设置高度，那么元素的高度就是行高。
  • 如果设置了 width: 100%，那么看起来就和块级元素一样。

清除浮动：clearfix

为所有float元素的父容器添加clearfix类
