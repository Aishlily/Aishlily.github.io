发表于 2018-01-02 | 分类于 CSS技术

通用字体族

  • serif，是一种有比例有衬线字体，有比例指的是字母宽度不一，有衬线指的是意思是在笔画开始、结束的地方有额外装饰，且粗细有所不同，常见字体有宋体、Times。
  • sans-serif，是一种有比例但无衬线字体，常见字体有黑体、Arial。
  • monospace，是等宽的sans-serif，用于模拟打字机打出的字体，常用字体是Courier，容易对齐。
  • cursive，书写体，用来模拟人手写笔迹，比如中文的华文行草。
  • fantasy，很少使用。

字体属性

  • font-size 字体尺寸，字体大小必须是12px以上，chrome最小支持12px字体
  • font-style 字体风格，normal、italic、oblique

  • font-weight 字体粗细

  • line-height 行高，单位: px、em、%
    一般而言：

  1. 如果使用的是像素px，那么子元素行高继承静态值不变；
  2. 使用百分比也和像素差不多，比如父元素字体为16px，行高120%，那么子元素实际继承的行高总是16*1.2=19.2px（约为20px），而和子元素的字体大小无关；
  3. 使用无单位数字，如果数值较大（如1.5），子元素继承到的也是相对数字1.5然后再乘以自己的字体大小（也就是和子元素字体大小相关），那么假定标题字体为30px，行高计算为30px*1.5=45px，会导致标题太松。
  • 可以把多个属性合在一起定义：只有 font-size 和 font-family 是必须明确定义的

文本属性

  • 文本对齐text-align属性必须用于块级元素，定义文本以及内联子元素如何横向对齐

    最常用的取值为：

    • left：把文本排列到左边。默认值：由浏览器决定。
    • right：把文本排列到右边。
    • center：把文本排列到中间。
    • justify：实现两端对齐文本效果。

text-align 的默认值是 start。start 可以是 left 或 right，这取决于HTML文档方向（direction）属性的设置。
direction 是一个CSS属性，可以是 ltr (left to right)或 rtl (right to left)：
如果 direction 等于 ltr，那么 start 等于 left
如果 direction 等于 rtl，那么 start 等于 right

  • 文本修饰text-decoration

    上划线（overline）
    下划线（underline）
    中划线或叫删除线（line-through）

  • 文本缩进text-indent，用来在块级元素的第一行添加一定的空格，以达到排版缩进的效果

  • 文本阴影text-shadow

  • 字母间隔letter-spacing

  • 单词间隔word-spacing
  • 处理空白符white-space
  • 处理文本大小text-transform
