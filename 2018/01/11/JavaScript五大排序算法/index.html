JavaScript五大排序算法

发表于 2018-01-11 | 分类于 算法

冒泡算法

比较任意两个相邻的项，如果第一个比第二个大，则交换顺序

选择排序

找到数组中最小的项并将其放到第一位，找到第二小的值，并将其放到第二位，依次……

插入排序

每次只排序一个数组项，确定它应该插入到哪个位置

归并排序（分治）

将数组拆分成较小的数组，直到每个数组的长度为1；
合并和排序小数组，直到回到原始数组的长度；

快速排序（分治）

  1. 从数组中选择中间项目作为主元
  2. 建立左右两个数组，分别存储左边和右边的数组
  3. 利用递归进行下次比较
